## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI user input `<path>` → AppContext | Path from CLI arg is a PathBuf; passed to convert_repo as &Path, canonicalized inside service |
| ConvertTarget enum → core service | CliConvertTarget converted to workpot_core::ConvertTarget in the match arm; no string interpolation |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07.1-04-01 | Tampering | `<path>` CLI arg → git subprocess | mitigate | Path is a `PathBuf` parsed by clap (typed, not raw string); passed to `ctx.convert_repo(&path, ...)` which canonicalizes before passing to git subprocess as separate `Command::arg()` call — never shell-interpolated |
| T-07.1-04-02 | Spoofing | `--to` value enum | accept | clap ValueEnum restricts to "bare" or "normal"; any other value fails with usage error before reaching any service code |
| T-07.1-04-SC | Tampering | npm/pip/cargo installs | accept | No new crates; clap already in workpot-cli Cargo.toml |
</threat_model>
