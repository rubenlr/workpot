## Threat Model Summary

| ID | Pattern | STRIDE | Mitigation |
|----|---------|--------|------------|
| T-path-traversal | Template with `../` resolves outside parent dir | Tampering | Validate resolved target paths are children of source's parent directory; reject on violation |
| T-arg-injection | CLI `<path>` arg passed to git subprocess | Tampering | Pass paths as separate `Command::arg()` calls, never shell-interpolated |
| T-rename-collision | Convert run twice, `{original}.temp` already exists | Spoofing | Pre-check for `{original}{temp_suffix}` existence; abort with actionable error before rename |
