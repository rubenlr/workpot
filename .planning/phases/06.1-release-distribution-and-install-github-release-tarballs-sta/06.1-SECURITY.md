# Phase 6.1 — Security

> Per-phase security contract: threat register, accepted risks, and audit trail.

---

## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| GitHub release event → CI artifact jobs | Untrusted trigger payload and repo state drive release build/upload behavior | Release tag, checkout ref, artifact names |
| CI secret store → signing steps | Apple signing credentials cross into DMG build runtime | `APPLE_*` secrets, signing identity |
| GitHub Releases API → install/update clients | Downloaded tarballs and DMGs are untrusted until checksum-verified | Release assets, `.sha256` sidecars |
| User filesystem → install/update targets | Global paths require elevated write access | CLI binary, `.app` bundle |
