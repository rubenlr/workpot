- **Fix:** Updated parent writability detection to walk up to the nearest existing parent and only escalate when truly required.
- **Files modified:** `scripts/install.sh`

- **Committed in:** `aed09de`

- **Found during:** Task 2 (smoke rerun after first fix)

- **Fix:** Promoted trap-tracked temp variables to non-local scope in installer and smoke scripts.
- **Files modified:** `scripts/install.sh`, `scripts/tests/install_smoke.sh`
