| Boundary | Description |
|----------|-------------|
| Watch root → filesystem walk | User-configured directories traversed read-only |
| Repo path → git subprocess | Canonical path passed as fixed -C argument (no shell) |
| Paths → SQLite | User paths bound via params! only |
