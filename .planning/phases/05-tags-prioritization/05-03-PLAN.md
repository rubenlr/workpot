    - `sectionSort(repos: RepoDto[], config: SectionConfig, nowSeconds: number): SectionedRepos`:
      - pinned: `repos.filter(r => r.pinned).sort((a,b) => (a.pin_order ?? 999) - (b.pin_order ?? 999))`
      - nonPinned: `repos.filter(r => !r.pinned)`
      - dirty: `nonPinned.filter(r => r.is_dirty === true).sort(byLastOpenedDesc)`
      - nonDirty: `nonPinned.filter(r => r.is_dirty !== true)`
      - windowSecs = `config.maxRecentDays * 86400`
      - recentByTime: `nonDirty.filter(r => r.last_opened_at != null && nowSeconds - r.last_opened_at < windowSecs).sort(byLastOpenedDesc)`
      - Pad recentByTime to `config.minRecentCount`: if `recentByTime.length < minRecentCount`, take additional repos from nonDirty (sorted by `last_opened_at DESC` with nulls last) not already in recentByTime until length = minRecentCount or nonDirty exhausted
      - rest: nonDirty repos not in recent set, sorted by `name` ascending (alphabetical)
      - Returns SectionedRepos object

    - Test: `sectionSort` with pinned repo appears in pinned only
    - Test: dirty repo appears in dirty (not recent/rest even if recently opened — D-20)
